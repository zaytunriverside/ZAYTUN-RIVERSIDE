Kenapa harus connect ulang tiap ganti printer/HP baru?
Itu wajar — begitu Anda ganti device atau printer, browser belum pernah dapat izin untuk device yang baru itu.

Kalau auto-reconnect tidak jalan (harus klik "Hubungkan" tiap buka halaman baru), kemungkinan versi Chrome di HP Anda belum mendukung fitur permission tersimpan. Coba cek di address bar Chrome: ketik chrome://flags/#enable-web-bluetooth-new-permissions-backend → kalau ada opsinya, aktifkan → restart Chrome → coba lagi. Kalau tidak ada opsi itu sama sekali, berarti versi Chrome Anda belum mendukung — tidak apa-apa, tinggal klik "Hubungkan" manual tiap pindah halaman, prosesnya cuma 1 kali sentuh, tidak perlu pilih device lagi.
